SQL练习题

现有一数据库，包括以下4个表：学生表（student）、课程表（course）、成绩表（score）、教师信息表（teacher）。4个表的结构如下所示：

表4-1 student（学生表）
属性名 数据类型 是否为空 含义
Sno Char(3) 否 学号（主键）
Sname Char(8) 否 姓名
Ssex Char(2) 否 性别
Sbirthday Datetime 是 出生日期
class Char(5) 是 所在班级
  • 此表的创建
表4-2 course（课程表）
属性名 数据类型 是否为空 含义
Cno Char(5) 否 课程号（主键）
Cname Varchar(10) 否 课程名
tno Char(3) 否 教师编号（外键）
  • 此表的创建
表4-3 score（成绩表）
属性名 数据类型 是否为空 含义
Sno Char(3) 否 学号（外键）
Cno Char(5) 否 课程号（外键）
Degree Decimal(4,1) 是 成绩
主键：sno+cno
  • 此表的创建
表4-4 teacher（教师信息表）
属性名 数据类型 是否为空 含义
Tno Char(3) 否 教工编号（主键）
Tname Char(4) 否 教工姓名
Tsex Char(2) 否 教工性别
Tbirthday Datetime 是 教工出生日期
prof Char(6) 是 职称
depart Varchar(10) 否 教工所在部门
  • 此表的创建
